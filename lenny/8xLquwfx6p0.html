增长停滞别瞎忙：两度独角兽创始人 Jason Cohen 的五步诊断与反直觉建议

01. 高层摘要

  • Jason Cohen 是四次创业者（含 WP Engine 等两只独角兽）、60+ 创业公司投资人，并在 A Smart Bear 上持续写作近二十年；本期把「产品不再增长」当成可系统拆解的工程问题，而非玄学。
  • 核心是一套五步诊断：先看 logo 留存，再查定价是否挡路，接着看 NRR（净收入留存）是否健康，再审视营销渠道是否饱和或错配，最后回到目标市场是否选对人。
  • 节目强调取消理由里「太贵了」往往是表层借口；真正原因常藏在价值感知、上手与激活、替代方案或关系断裂里，需要更好的取消调研与定性反馈机制。
  • 一个小改动可让取消问卷回复率约翻倍（节目宣称量级），说明流失环节上的「提问方式」本身就会影响你能否拿到真因。
  • 「大象曲线」描述增长通道往往先陡后平、渠道终会衰减；需区分直销与间接渠道，并主动寻找新通道，而不是假设某一渠道永远线性放大。
  • 同一产品通过重新定位（positioning）带来约 8 倍收入的故事被用作提醒：有时瓶颈不是功能，而是你说服谁、为谁解决什么问题。
  • 反共识的一刀：并非所有业务都应以「持续高速增长」为默认目标；需要追问这是否对客户真的好、现有客户是否已被吃干榨尽，以及何时应认真考虑退出。

02. 本期主题与嘉宾背景

嘉宾：Jason Cohen

公司：WP Engine（及多次创业与投资履历）

角色：创始人、投资人、长期写作者

背景：四次创业经历，其中包含两只独角兽公司（其一为 WP Engine）；投资超过 60 家初创企业；以 A Smart Bear（及 longform.asmartbear.com）分享公司构建与增长方法论近二十年；本期关联其新书预售（Hidden Multipliers 相关页面）。

本期主题：当产品增长停滞时，如何用五步框架（留存、定价、NRR、渠道、目标市场）做根因诊断；取消与流失调研、上手激活、定价与战略的一体性、渠道饱和与「大象曲线」、定位叙事对收入的杠杆，以及何时质疑「增长」本身是否为正确目标。

为什么重要：兼具规模化 SaaS 实战与大量投后观察的少数声音之一；把增长停滞从「再投一波营销」还原为可排序的假设检验，对 B2B、订阅制与任何重复购买业务都有对照价值。

适合团队：增长与留存、产品与激活、定价与商业化、市场与品牌定位、客户成功与销售、数据分析、战略与投融资沟通

03. 核心观点精读

观点 1：五步框架：把「不增长」拆成可验证的五层

核心观点

几乎每支团队都会遇到增长平台期；与其泛泛优化，不如按顺序检视：logo 留存 → 定价 → NRR → 营销渠道 → 目标市场，避免在错误层级上用力。

关键论据

节目大纲明确列出 Jason 的五步框架，并将其作为诊断停滞增长的主线。

典型案例

适用于订阅/续费业务：若留存未稳却猛砸获客，往往在数学上不可持续；框架强迫你先修「漏桶」再谈放大。

对公司的启发

内部复盘可按五步做「红绿灯」：每层对应 1–2 个主指标与下一轮实验，而不是所有团队同时抢同一笼统 KPI。

观点 2：取消调研：「太贵」很少是真因，提问方式决定数据质量

核心观点

客户口头上的价格异议，常与真实动机脱节；若问卷设计默认引导到价格，你会系统性误判产品与市场。

关键论据

Show notes 单列讨论：为何「太贵」几乎从不是取消的真正原因；并提到一个可显著提升取消问卷回复率的小改动。

典型案例

流失分析可参考节目引用资源（如 Groove 关于 churn analysis 的文章）所代表的行业共识：要结合行为、使用深度与定性访谈。

对公司的启发

对高客单或合约业务，应用「退出访谈 + 行为数据」双轨；避免在取消流程里用封闭式选项把用户推向唯一「体面借口」。

观点 3：上手与激活：增长停滞有时是「买进来却用不起来」

核心观点

获客之后若激活路径断裂，表面像「渠道不行」，实则是产品内首次价值交付失败。

关键论据

时间轴包含专门段落讨论 onboarding 与 activation，并与「产品停止增长时要问的问题」前后呼应。

典型案例

可与 Duolingo、Grammarly 等「重燃增长」类节目对照：许多二次增长来自核心循环与习惯的修复，而非单纯新增功能。

对公司的启发

为激活定义「时间盒内完成的关键事件」；在增长评审中与渠道花费并列，避免 CAC 优化掩盖激活漏斗恶化。

观点 4：定价不是独立旋钮，而是战略的一部分

核心观点

价格传递你是谁、服务谁、替代了谁；与包装、销售动作、目标细分强耦合，不能与「战略」分开优化。

关键论据

节目多次回到定价策略，并专章讨论「定价为何与战略不可分」；引用链中含 Madhavan Ramanujam 等多期 Lenny 定价专题与 Patrick Campbell 等外部观点。

典型案例

企业销售与「卖 alpha 不卖功能」等 playbook 被列作延伸材料，暗示定价叙事与售卖叙事同一套。

对公司的启发

重大定价变更应同步：ICP（理想客户画像）、合约条款、销售激励与客服话术；否则会出现「战略上升级、执行上拆台」。

观点 5：NRR：健康扩张的引力中心

核心观点

净收入留存连接扩张质量：靠降价换来的增长与靠扩展用例换来的增长，长期曲线完全不同。

关键论据

Show notes 强调 NRR 的重要性，并与 leveraging existing customers（深挖存量）形成同一叙事支线。

典型案例

高 NRR 业务更易承受渠道波动；低 NRR 时加码投放往往放大亏损客户占比。

对公司的启发

将 upsell/cross-sell、用量计费、席位扩展与流失挽回纳入同一「收入留存」看板，而不是只盯新签 ARR。

观点 6：「这对客户好吗？」——伦理与商业的交叉校验

核心观点

增长手段若损害客户结果，短期指标可能仍好看；长期会反噬口碑与留存。

关键论据

时间戳专设「Asking whether or not this is good for the customer」，与 Jason 长期写作中强调可持续、诚实营销的风格一致。

典型案例

适用于暗促、模糊续费、过度挤压使用限制等灰色增长；问题不是「能不能做」，而是「做了客户会不会更快走」。

对公司的启发

在上线增长实验清单中增加「客户结果」一票否决项：对关键细分可追踪 outcome 指标，而非仅转化率。

观点 7：大象曲线与渠道必然衰减

核心观点

营销渠道往往呈现早期陡峭、随后趋缓甚至下滑的轨迹；「所有渠道最终都会走弱」被作为常态而非意外。

关键论据

节目讨论获客渠道是否饱和、介绍「elephant curve」，并引用 Jason 长文《The Elephant in the room: The myth of exponential hypergrowth》；另设「直销 vs 间接渠道」「创意寻找新渠道」段落。

典型案例

与 HubSpot、Buffer 等案例一同被列入参考，提示渠道组合需迭代而非单押。

对公司的启发

建立「渠道生命周期」复盘：对每个主渠道记录边际 ROI 与饱和度信号，提前 1–2 个季度布局下一曲线。

观点 8：定位重叙事：同一产品可带来数量级收入差异

核心观点

当你对谁说、解决什么高 stakes 问题变了，支付意愿与成交效率会变；产品代码未必大改。

关键论据

Show notes 明确提到 repositioning 同一产品或带来约 8x 收入；并有独立「Positioning strategies」章节。

典型案例

适合陷入「功能完备却卖不动」的团队：用 ICP 与替代方案重新定义品类故事，再反推包装与案例。

对公司的启发

每半年做一次「定位审计」：竞品话术、客户原话、赢单/输单理由是否仍与首页叙事一致。

观点 9：增长不是天职：何时质疑目标、何时考虑退出

核心观点

并非每个生意都应无限追求规模；有时现金流业务、并购整合或主动收缩更理性。

关键论据

节目后段直接讨论「你真的需要增长吗？」「何时该 quit」，与多数增长播客形成反差，也是标题中 surprising advice 的来源之一。

典型案例

对小型盈利业务或赛道天花板清晰的团队，盲目融资扩张可能摧毁单位经济。

对公司的启发

董事会与创始人应用「战略选项表」并列：维持、深耕细分、并购出售、关停产品线，而不是默认下一轮融资叙事。

04. 方法论提炼

Framework核心问题使用场景关键步骤参考价值
增长停滞五步诊断（Jason Cohen）瓶颈是在留住人、卖对人、卖贵/卖便宜、扩张质量，还是渠道与市场的错配？月活/收入增速放缓、销售周期拉长、投放效率下滑但原因不明时。1）logo 留存与核心使用是否健康。2）定价与包装是否阻断正确客户或吸引错误客户。3）NRR 是否支撑可持续扩张。4）主渠道是否进入大象曲线尾部。5）目标市场与定位是否仍成立。把「增长团队加预算」从默认答案变成排序最后的手段，优先修结构性漏损。
取消与流失：真因访谈清单用户离开时说的理由，与行为与价值感知是否一致？流失率上升、客诉集中在价格、竞品切换增多时。1）优化取消流中的问卷与访谈，提高回复率。2）交叉比对最后 30 天行为与 support 记录。3）对「太贵」单独做价值/替代方案探针。4）输出可行动的三大根因假设并验证。减少把定价当万能借口，避免错误降价侵蚀品牌与 NRR。
大象曲线 × 渠道组合当前主渠道处于曲线的哪一段？下一增长曲线在哪里？单一渠道占比过高、CPA 攀升、行业流量红利消退时。1）为每个渠道画时间与边际获客成本。2）区分直销与间接，评估可规模化上限。3）设「渠道实验预算」探索新通路。4）与定位调整联动，避免渠道与叙事脱节。用组合思维对抗「指数增长永远持续」的幻觉。

06. 公司级行动清单

行动项适用团队适用场景预期收益优先级
用五步框架开一次增长诊断会：每层指定负责人、当前指标、下一项验证实验与截止日期。增长负责人 / 产品负责人 / 数据增速连续两个季度低于预期避免全员只盯投放或只盯功能，快速定位主瓶颈层级P0
重做取消/销户问卷与退出访谈：对照节目中「提升回复率的小改动」思路，减少引导性价格选项，增加开放式与行为核对问题。用户研究 / 客服 / 产品流失率上升或竞品切换增多提高真因命中率，指导定价与产品路线P0
为激活定义「首周关键事件」仪表盘，与获客花费在周报中并排展示。产品 / 增长获客成本升高但留存未改善识别「买进来却用不起来」类问题P1
定价与包装变更前做一次「战略一致性」评审：ICP、销售激励、合同模板与客服脚本是否同步更新。商业化 / 销售 / 法务计划调价、改套餐或改佣金减少执行层拆台与客户预期混乱P1
建立 NRR/扩展收入看板：按细分用户群拆解 upsell、交叉使用与流失挽回贡献。财务 BP / 数据 / 客户成功收入增速依赖新客、老客贡献不透明把扩张质量可视化，支撑资源分配P1
绘制主获客渠道的「时间 × 边际成本」曲线，并设不少于 10% 预算用于新渠道实验。市场 / 增长单一渠道占比超 50% 或 CPA 持续上升缓解渠道饱和与大象曲线尾部风险P1
半年度定位审计：对比竞品首页、客户访谈原话、赢单理由，修订核心叙事与案例库。市场 / 产品 / 销售功能完备但成交效率低释放定位杠杆，避免只会堆功能P2
高管工作坊：列出「维持 / 深耕 / 并购退出 / 收缩」选项，明确何种指标下应切换叙事，而非默认永远追求超高增长。CEO / 战略 / 董事会赛道天花板清晰或资本环境收紧减少路径依赖与无效融资故事P2

07. 讨论问题

1. 我们当前最大瓶颈在五步中的哪一层？如果只能修一层，团队共识是哪一层？

2. 取消/销户理由里，有多少经过与行为数据的交叉验证？最近一次因「太贵」做的决策是否被数据支持？

3. 主渠道是否已出现大象曲线拐点？下一渠道实验的假设与衡量标准是什么？

4. 定价与包装是否仍在传达「我们服务谁」？若换一类核心客户，叙事要改哪些句子？

5. 若不以规模为唯一目标，我们更健康的产品与组织形态是什么？何时应认真讨论退出或合并？

09. 一句话总结

一句话总结：增长停滞时，先用「留存—定价—NRR—渠道—市场」五步定位真问题；警惕取消理由里的价格借口与渠道的「大象曲线」，并敢于问：更好的定位与更诚实的目标，可能比再加一波投放更有效。
